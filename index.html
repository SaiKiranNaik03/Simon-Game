Simon Game

Press any key to start!

Rules:

The Simon game is a classic memory game that challenges players to repeat a sequence of colors and sounds. Here are the rules:

Start of the Game:The game begins by lighting up one of the four colors (red, blue, green, yellow) along with a sound.

Player's Turn:The player must repeat the pattern by clicking the colors in the exact same order.

Progression:After each successful attempt, the game adds one more color to the sequence, making it progressively harder.

Mistakes:If the player clicks the wrong color or sequence, the game ends, and the player has to start over (depending on how you implement this).

Difficulty:The sequence continues to get longer as long as the player correctly follows the pattern.

Goal:The aim is to memorize and correctly repeat as long a sequence as possible without making a mistake.
